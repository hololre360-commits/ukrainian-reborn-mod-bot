# Ukrainian Reborn — S-tier Moderation Bot

Преміум бот модерації спеціально для спёльноти **Ukrainian Reborn** (RP сервер Emergency Hamburg, Roblox).

Аналог GroupHelp + Rose з власними фішками преміум-рівня та **максимальним керуванням прямо з Telegram**.

## ✨ Ключові переваги

- Повністю керується з Telegram (команди + красиві inline-клавіатури)
- Розумна авто-ескалація попереджень (warn → mute → ban)
- Стильний captcha при вході з кнопкою підтвердження
- Inline-панель обробки скарг у лог-каналі (S-tier UX)
- Авто-фільтри + anti-flood
- Кастомні модератори (не тільки Telegram-адміни)
- Видалення службових повідомлень (join/leave)
- RP-specific типи порушень
- Чиста архітектура, швидкий код, prepared statements + кеш

## 🚀 Швидкий старт

```bash
cd ukrainian-reborn-mod-bot
npm install
cp .env.example .env
# Відредагуй .env і встав свій BOT_TOKEN
node bot.js
```

## Налаштування в Telegram

1. Додай бота в supergroup Ukrainian Reborn
2. Зроби його **адміністратором** з правами:
   - Видалення повідомлень
   - Блокування користувачів
   - Закріплення повідомлень (опціонально)
3. Використовуй `/setlog` (перешли повідомлення з приватного каналу для логів)
4. Налаштуй `/togglecaptcha`, `/toggleservice` тощо
5. Додай кастомних модераторів через `/addmod` (тільки creator)

## 🔥 Власні круті фішки (які я додав)

1. **Inline Report Panel** — користувачі тиснуть `/report` → модератори отримують в лог-канал повідомлення з кнопками дій (Warn/Mute/Kick/Ban/Delete/Dismiss). Одна з найкращих фішок преміум-ботів.
2. **Розумна ескалація warns** — при досягненні 75% від max_warns → авто-мут, при 100% → бан. Налаштовується.
3. **RP Violation tracking** — тип `rp_violation` для окремої статистики (можна розширити).
4. **Captcha з кнопкою** — красивий, простий, без складних математичних завдань.
5. **Anti-flood + word filters** — працює в реальному часі.

## 🛠 Технології (кращі практики)

- **Telegraf 4.x** — сучасний, швидкий, зручний
- **better-sqlite3** — найшвидша вбудована БД для Node.js
- In-memory cache для налаштувань
- Prepared statements + WAL mode
- Graceful error handling + logging
- Чіткий поділ коду на секції

## ⚠️ Важливо

- Бот потребує Node.js >= 18
- Для production використовуй `pm2` або Docker
- Зберігай токен у .env (ніколи не коміть)
- При великій кількості чатів можна розширити на PostgreSQL

## Підтримка

Бот створений спеціально для Ukrainian Reborn з фокусом на чистому коді, швидкості та зручному UX для модераторів і учасників.

Якщо потребні додаткові фічі (статистика, reputation system, інтеграція з Roblox API, веб-панель тощо) — пиши.
